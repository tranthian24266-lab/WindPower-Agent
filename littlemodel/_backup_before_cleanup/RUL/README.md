# Wind HSSB RUL Small Model Library

This folder packages five trained small models for wind turbine high-speed shaft bearing RUL work.

Source project:

- Dataset root: `C:\Users\luzian\Desktop\windpower_dataset\RUL\WindTurbineHighSpeedBearingPrognosis-Data`
- Reproduction project root: `C:\Users\luzian\Desktop\windpower_dataset\RUL\reproduction`

## Folder Layout

- `models/`: trained `.joblib` small models
- `test_data/split_60_40/`: chronological test set used by all `60/40` models
- `test_data/split_40_60/`: chronological test set used by all `40/60` models
- `metadata/predictions/`: per-model prediction CSV outputs
- `metadata/metrics_summary.csv`: consolidated evaluation metrics
- `metadata/final_summary.md`: summary generated by the reproduction pipeline

## Model Overview

### 1. `svr_kurtosis_baseline_60_40.joblib`

- Type: paper baseline small model
- Input features: `kurtosis`
- Split: first 60% train, last 40% test
- Test set: `split_60_40`
- Purpose: single-feature baseline for paper-style reproduction

### 2. `svr_area_sk_60_40.joblib`

- Type: paper reproduction main small model
- Input features: `sk_area`
- Split: first 60% train, last 40% test
- Test set: `split_60_40`
- Purpose: original paper-style single-feature `Area under SK + SVR`

### 3. `svr_area_sk_40_60.joblib`

- Type: paper reproduction control small model
- Input features: `sk_area`
- Split: first 40% train, last 60% test
- Test set: `split_40_60`
- Purpose: paper-style chronological control experiment

### 4. `svr_demo_multifeature_60_40.joblib`

- Type: demo small model
- Input features: `sk_area_positive`, `kurtosis`, `std`, `peak_to_peak`
- Split: first 60% train, last 40% test
- Test set: `split_60_40`
- Purpose: improve demo stability with multi-feature input while preserving the original feature extraction pipeline

### 5. `svr_demo_sk_multi_linear_40_60.joblib`

- Type: demo small model
- Input features: `sk_area`, `sk_area_positive`, `sk_max`, `sk_std`, `sk_peak_to_peak`, `sk_mean`
- Split: first 40% train, last 60% test
- Test set: `split_40_60`
- Purpose: demo-oriented multi-SK small model with linear or RBF SVR search enabled

## Notes For Agent-Based Management

- All splits are strictly chronological. No random split was used.
- The raw source data inside `test_data/` are copied from the official downloaded dataset and grouped by split for reuse.
- Prediction CSV files in `metadata/predictions/` record `true_rul`, `pred_rul_raw`, `pred_rul_clipped`, feature list, and train/test flags.
- `clipped` RUL values are only for engineering-safe display and demo output. They do not replace raw reproduction metrics.
- This library is based on a single 50-point run-to-failure sequence, so these models are suitable for reproduction and demo orchestration, not engineering-grade fleet deployment.
